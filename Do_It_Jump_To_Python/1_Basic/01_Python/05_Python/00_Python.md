## Install Python
- `https://www.python.org/downloads/`
- 3.7.4 version
- Add Python 3.7 to PATH
- Python 3.7 (32-bit) 실행 -> Python 인터프리터 실행. (Python Sell 이라고도 함.)
  - `CMD`에서 `py` 명령하면 실행됨.
  - `>>>`를 프롬프트 라고 함.
  - win : `ctrl + z -> enter` mac : `cmd + d`로 종료 할 수 있음.
  - 또는 `import sys  sys.exit()`
